AQUA Diagnostics
================

The dashboard presents diagnostic figures generated by `AQUA <https://aqua.readthedocs.io/en/latest/>`_ (Climate DT Applications for QUality Assessment), an open-source model evaluation framework for high-resolution climate simulations.

AQUA processes Climate DT simulation output and produces standardized diagnostic figures that assess model performance by comparing against observational datasets and reanalysis products such as ERA5. Diagnostics are generated automatically as simulations progress on EuroHPC systems.

Available diagnostic categories include Gregory Plots, Time Series, Seasonal Cycles, Performance Indices, Radiation, Bias Maps, Tropical Rainfall, Teleconnections, Ocean Time Series, Ocean Trends, Ocean Stratification/MLD, and Sea Ice. The set of available categories varies by model configuration.

For detailed information on the AQUA framework, diagnostics, and methodology, see the AQUA documentation:

- `AQUA documentation <https://aqua.readthedocs.io/en/latest/>`_
- `AQUA diagnostics documentation <https://aqua-diagnostics.readthedocs.io/en/latest/>`_
- `AQUA source code <https://github.com/DestinE-Climate-DT/AQUA>`_
- `AQUA diagnostics source code <https://github.com/DestinE-Climate-DT/AQUA-diagnostics>`_
