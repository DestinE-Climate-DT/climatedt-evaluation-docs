Accessing Climate DT Data
=========================

The data from Climate DT simulations visualized in this dashboard can also be accessed directly for analysis.

Polytope
--------

`Polytope <https://platform.destine.eu/services/service/polytope/>`_ is the primary service for accessing Climate DT data on DESP. It is an open-source web service for efficient access to hypercubes of scientific data, able to federate access between distributed computing resources. Users can access data via the REST API or the ``polytope-client`` Python package.

To access Climate DT data via Polytope:

1. Register on the `DestinE Service Platform <https://platform.destine.eu/>`_
2. Request `upgraded access <https://platform.destine.eu/access-policy-upgrade/>`_
3. Follow the `Polytope examples <https://github.com/destination-earth-digital-twins/polytope-examples>`_ to get started

AQUA also supports Polytope as a data access engine. See the `AQUA Polytope documentation <https://aqua.readthedocs.io/en/latest/polytope.html>`_ for details on using Polytope through AQUA.
